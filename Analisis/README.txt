Proyecto de Analisis de Algoritmos: Comparación de algoritmos de ordenamiento
Autores: Ginger Jacome, Leonardo Gomez, Henry Maticurena, Eduardo Tigse
Archivo principal: Analisis
Aspectos a tomar en cuenta:
- Las librerias que se necesitan para generar las gráficas se llaman jfreechart-1.0.19 y jcommon-1.0.23 las cuales se 
encuentran en la carpeta lib dentro del archivo principal.
- Existen dos formas de como observar los tiempos de ordenamiento de los algoritmos: 
	1. seleccionar un archivo de texto ya existente con los datos a ordenar.
	2. generando un archivo de texto con números aleatorios para posteriormente ordenarlo.
En el segundo caso, el usuario tendra la oportunidad de ingresar el número maximo de datos y el nombre del archivo.
- El número maximo de datos que pueden generarse aleatoriamente sin que el programa demore es de 10000, pasado ese 
limite el programa presenta una demora al momento de mostrar las gráficas.
- El usuario podra seleccionar los algoritmos a comparar al cambiar a la opcion de 'si' del menu de opciones que 
se encuentra al lado de cada algoritmo.
- El usuario podra observar la gráfica con los algoritmos seleccionados al darle clic al boton Iniciar.
- Los tiempos de ordenamiento de los algoritmos seleccionados se imprimen en el archivo de texto llamado resultados.
- En caso de no saber como colocar las librerias dentro del programa Analisis, dentro de la carpeta dist hay 
un ejecutable del programa.
